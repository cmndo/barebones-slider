Barebones Feature Slider

An unassuming feature slider plugin for jQuery

Finding a feature slider to use is way harder than it should be. There are a lot of them out here already and they all seem to be packed with features. If you don't want a lot of features, customizing one to your liking can be the most frustrating thing ever. Barebones is a fresh start.

How does it work?

Target the element that holds the content of your slides. Then

var example = $('.wrapper').barebones()

Then all you have to do is call example.next() and example.back() to move forward and backward

use jQuery events to listen for slideChange to detect when the slide changes.

example.on('slideChange', function(evt, slide){
	console.log("Changed to slide " + slide + ".");
});
